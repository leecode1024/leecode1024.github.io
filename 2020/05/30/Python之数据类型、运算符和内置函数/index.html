Python之数据类型、运算符和内置函数

发表于 2020-05-30 | 更新于: 2020-05-30 | 分类于 Python | 阅读次数:
字数统计: 2.8k | 阅读时长 ≈ 10

前言

本文介绍python的基本数据类型、容器类型区别和注意点、运算符的一些注意点，以及常用的内置函数。

运算符

这里介绍一下，python中运算符的特殊用法。

  • +不仅是加，字符串连接，也是列表，元组的连接
  • -减号，可求集合差集
  • 乘号，可使序列元素重复 例如[0]5 [0,0,0,0,0]
  • //求商，向下取整 17//4=4，-17//4=-5，结果是小于除法的最大整数。尤其注意负数情况。
  • **幂运算，指数可为小数，例如0.5代表开平方根
  • and or not 逻辑运算或与非，惰性求值。
  • 关系运算符【<,>,==,<=,>=,!=】运算优先级小于算数运算符。连续使用具有惰性求值特点。
  • 字符串，列表进行关系运算符的比较，是每个字符或元素进行比较，惰性求值。
  • 成员测试运算符in 一个对象是否包含另一个对象。
  • 集合运算符[&,|,^,-]分别为交集，并集，对称集，差集

常用内置函数

python提供了大量内置函数供我们使用，内置函数不需要导入任何模块，可以直接使用。

类型转换

  • int(),float(),complex()
    • int()将浮点数转化为整数，或将数字字符串按照指定进制转化为十进制。如果不指定进制，则直接转为十进制。int(3.5),int(‘123’),int(‘111’,2)。
    • float()将其他数据类型转化为浮点数。
    • complex()用来生成复数。complex(3,4) out:(3+4j),complex(‘3’) out:(3+0j)
  • bin(),oct(),hex()将任意进制整数转化为2,8,16进制数。
  • ord(),chr(),str()
    • ord()用于返回字符的unicode编码，chr()用于返回对应unicode编码的字符，两者为逆运算。
    • str()将任意类型整体转为字符串。
  • list(),tuple(),dict(),set()强制转换为列表，元组，字典，集合或者创建空对象。
  • eval()用来计算字符串或字节串的值，也可以用来还原数据的实际类型，例如输入时可以直接进行类型转换。

最大值、最小值

max(),min()用于返回最值，参数key可以指定排序规则。例如：max(data,key=len)返回data中len最大的元素。

元素求和

len()计算序列长度，sum()计算序列元素之和，一般要求类型一致。

排序、逆序

  • sorted(data) 可对列表，元组等对象排序，并返回新对象，支持key指定排序规则，reverse=False为升序True为降序，默认升序。
  • reversed()翻转并返回reversed对象，具有惰性求值特点，即只能用一次。

基本输入输出

  • input()接收用户键盘输入，一律返回字符串。可以配合类型转换函数使用。
  • print(v1,v2,v3...,sep=' ',end='\n')其中v1,v2..为要输出的内容，sep指定用什么分隔，end指定用什么结尾。

range()

range(开始，结束，步长),左闭右开，惰性求值，可转换为列表，元组或集合

zip()

用于将多个迭代对象中对应位置上的元素组合在一起，返回一个可迭代的zip对象

data=zip(‘1234’,[1,2,3,4,5,6])
list(data)

out:[(‘1’,1),(‘2’,2),(‘3’,3),(‘4’,4)]

map(),reduce(),filter()

  • map()将一个可调用对象func依次映射到序列的每个元素上，并返回一个可迭代对象。
    • 例如：list(map(str,range(5))) [‘0’,’1’,’2’,’3’,’4’]
  • reduce()注意这不是内置函数！！而是在functools中，用之前需要导入！！
    • reduce(func,seq[,inital])
    • 作用：可以将一个接收2个参数的函数以迭代的方式从左到右依次作用到一个序列或可迭代对象上，中间结果参与下一次运算，最后得到一个值。
    • 例如：seq=range(1,10) print(redece(add,seq)) 结果为：45
  • filter()使用指定函数对序列元素进行过滤。
    • filter(func or None,iterable)
    • filter将一个func作用于序列上，返回filter对象，其中包括func返回值为True的元素
    • filter将None作用于序列上，返回filter，其中包括原序列中等价为True的元素。

基本数据类型

对象类型 类型名称 示例 说明
数值 int、float、complex 8888888、9.8、5+6j 数值大小没有限制
字符串 str ‘asdasdasd’、r’C:\windows’ 使用单、双、三引号为定界符，前面加r表示不转义

字符串常用方法

  • encode(),decode()

    • encode()指定编码格式对字符串编码为字节串，默认UTF-8
    • decode()使用指定编码格式对字节串解码为字符串，与上方为逆运算
    • 注意，编码与解码格式需要一致。

  • index().rindex(),count()

    • index()返回一个字符串在当前字符串中首次出现的位置，不存在会抛异常
    • rindex()返回一个字符串在当前字符串中最后一次出现的位置，不存在会抛异常
    • count()返回一个字符串在当前字符串中出现的次数，不存在则返回0

  • replace(),maketrans(),translate()

    • replace()用来替换字符串中指定字符或字符串的所有重复出现。相当于word里的全部替换，返回新字符串，不改变原字符串

    • maketrans()用于生成字符映射表，translate()根据映射表中分关系进行映射。

  • ljust(len,填充符),rjust(),center()用于对字符串进行排版，返回指定宽度的新字符串，可使用指定字符进行填充，默认为空格

  • split(),rsplit(),join()

    • split()使用指定字符串作为分隔符（默认空格）对原字符串从左到右进行分隔，返回分隔后的字符串列表。
    • rsplit()也是进行字符串分隔，不过是从右向左
    • split()与rsplit()可用参数maxsplit指定最大分隔次数。
    • join()使用指定字符串作为连接符，对若干字符串进行连接。
      • 例如：print(‘,’.join([‘a’,’b’,’c’,’d’])) 输出：a,b,c,d

  • lower(),upper(),capitalize(),title(),swapcase()

    • lower(),upper()将字符串中英文字符转化为小、大写字母
    • capitalize()每个句子的第一个字母大写
    • title()每个单词的第一个字母大写
    • swapcase()大小写互换

  • startwith(),endswith()判断字符串是否以指定字符或字符串开始或结尾。

容器数据类型

容器数据类型包括：列表、元组、字典和集合。

列表

  • [1,2,3,4,5,6]，所有元素放在一对方括号中，相邻元素之间使用逗号分隔。

  • 列表中元素的数据类型可以不同。

  • list()可以将其他类似对象转换成列表

  • 列表下标访问支持双向索引，范围[-L,L-1],L为长度。正向索引0代表第一个元素，反向索引-1代表最后一个元素。

列表常用方法

注意：这里对于列表的增删改，排序等操作，都是在原对象上进行的，没有返回值或返回None！

列表推导式

例如：data=[num for num in range(20) if num%2==1]

如果包含多个for循环，越靠前的for循环，越在内部

切片

[开始:结束:步长]，左闭右开，step默认为1

元组

元组跟列表类似，可看作轻量级列表。

元组和列表的区别

  • 元组不可变，不能对元组的元素进行增删改
  • 元组的访问速度更快，开销更小。
  • 元组可作为字典的键或集合的元素。

生成器表达式：

gen=(2**i for i in range*(8))

tuple(gen) 转换为元组

生成器都满足惰性求值，可以逐个访问，但不可后退

字典

{

key1:value1,

key2:value2….

}

创建字典

data=dict(name='张三',age=18)

data=dict.formkeys([1,2,3,4]) //指定键，值为空

data={ch:ord(ch) for ch in 'abcd'} //字典推导式

元素访问

get()方法用于获取指定键对应的值，如果不存在get()方法会返回空值或指定的值
get(key,’指定值’)

字典的修改，删除与添加

  1. 以指定键为下标为字典元素赋值，若该键存在，则修改；若不存在，则新增。
  2. update()将另一字典一次性添加到当前字典，若键重复，则根据后者更新。
  3. pop()删除指定键的元素，返回对应的值。popitem()删除并返回包含两个元素的元组，分别为key,value
  4. del sock[‘key’]删除指定键对应的元素

集合

无序、可变容器对象，所有元素放在一对大括号中，逗号分隔，元素唯一，不能重复。

常用方法

  • add(),新增元素，已存在则忽略，不抛异常。
  • update()合并另一个集合的元素到当前集合，自动去重。
  • pop()随机删除并饭后集合中的一个元素，集合为空则会抛异常。
  • remove()删除指定元素，不存在则会抛异常
  • discard()删除指定元素，不存在则忽略该操作。

空对象创建注意：

空列表：s=[]或s=list()

空元组：t=[]或t=tuple()

空字典：s={}或s=dict()

空集合：因为集合与字典都是包含在{}内，空字典可以用{}声明，而空集合不行，空集合只能用s=set()

包含列表的元组

元组是不可变的，但列表是可变的，如果一个元组中有列表，那么这个列表改变，是不是这个元组也改变了？

其实没有改变。Python中变量不直接存储值，而是存储值的内存地址或引用。
元组中有列表，则存储的是该列表的地址或引用。列表中有值，也是存储的该值的地址或引用。列表改变，改变的是列表中值的地址，而列表的地址并没有改变，所以元组中存的内容也就没有变化。

注意：元组可以作为字典的键或者集合的元素，但是包含列表的元组不可以！！！
