Python之选择、循环结构，函数的定义与使用

前言

这里介绍python中的选择、循环结构的各种表现形式，以及python特有的语法。简单介绍python中函数的使用，函数参数的各种类型等等。

选择结构

条件表达式

  • 所有合法表达式都可为条件表达式。
  • 条件表达式的值为True和False
  • 数字作为条件表达式0,0.0,0j等价于False，其他任意数字均等价于True
  • 不含任何元素的容器类对象等价于False
  • 不含任何字符的空字符串等价于False，哪怕有个空格都为True

循环结构

for循环

python为循环结构加入了else块，else块在循环体自然结束后执行。如果循环体提前结束，即执行了break语句，则不执行。

while循环

break与continue

这里的作用跟C语言，java语言作用一样。

  • break 直接结束循环体
  • continue 跳过之后的循环体语句，进入下次循环

函数

函数的定义

  • def 是定义函数的关键字
  • 参数不需要指定类型，python自动判断
  • 不需要说明返回值类型，可以返回多个值，没有返回值，认为返回None

lambda表达式

  • 等价于一个函数，用于需要一个函数，但又不想定义的场合。

  • lambda表达式只能包含一个表达式，不允许包含选择、循环等语法结构。

  • 格式：lambda [参数列表]:表达式

  • 多个参数用逗号隔开，没有参数，表达式的值等于函数的返回值。

递归函数

  • 一个函数在执行过程中，又调用本身。

生成器函数

  • 函数中包含yield语句，代码每次执行到yield语句时，返回一个值，然后暂停执行，当通过其他方式索要数据时，在恢复执行。

  • 也就是说yield会返回一个值，但返回之后会记住执行到哪里了，当再次索要数据时，会继续执行。

  • 返回的生成器对象具有惰性求值的特性。

参数

  • 位置参数

    • 即：实参与形参，在位置与数量上对应。这里跟C语言，java语言用法一样。

  • 默认值参数

    • 在定义函数时，为部分形参设置默认值。
    • 在调用函数时，如果不为已经设置了默认值的参数传值，则使用默认值。如果传了值，则使用实参的值。
    • 在语法上，任何带默认值的参数，其右边不能有不带默认值的参数。

  • 关键参数

    • 关键参数是指，明确给哪个参数传值，类似于key=value。

    • 这时候，如果参数位置不对应，也无所谓。

  • 可变长度参数

    • 可变长度参数指：一个形参接收实参的数量是可变的

    • 形参前加一个*，表示接收多个位置参数，并把他们放到一个元组中

    • 形参前加两个*，表示接收多个关键参数，并放到字典中。

变量作用域

这里介绍局部变量和全局变量

  • 在函数中只有引用变量值，而没有赋值的操作，该变量叫全局变量
  • 如果在函数中有为变量赋值，则该变量为局部变量，除非在操作变量前用global进行声明。
  • 局部变量与全局变量相同，局部变量会覆盖全局变量。
